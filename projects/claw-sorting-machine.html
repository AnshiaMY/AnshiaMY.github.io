This project combined mechanical assembly, control logic, testing, and iteration to create a VEX-based sorting machine with a functional claw mechanism.

Testing & Debugging

Challenges and Improvements

Testing helped reveal how small mechanical and control changes affected the robot’s ability to complete the sorting task reliably.

Grip consistency was important

The claw needed to apply enough force to hold objects without slipping or pushing them out of position.

Alignment affected performance

Small changes in object position or robot angle could affect whether the claw picked up the object successfully.

Mechanical structure affected control

The physical build, motor placement, and claw geometry all influenced how smoothly the mechanism moved.

Final Takeaway

Final Reflection

This project helped me understand how mechanical design and control logic work together in a physical robotics system. The most valuable part was testing the claw mechanism and seeing how real-world factors like alignment, grip, and structure affected performance.

Future Improvements

  • Improve claw grip consistency
  • Make object alignment more reliable
  • Strengthen the mechanical structure
  • Refine timing and control logic for smoother sorting
