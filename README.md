## About This Portfolio

I am a Mechatronics Engineering student interested in building systems where programming, electronics, and mechanical design work together.

This portfolio documents both the final results of my projects and the engineering process behind them, including:

- Design decisions and technical trade-offs
- Hardware and software integration
- Testing and iteration
- Debugging and troubleshooting
- Final reflections and future improvements

The portfolio is designed to support applications for opportunities in software, robotics, embedded systems, and automation.

## Featured Projects

### Autonomous Line-Following Robot

A Raspberry Pi-based mobile robot that uses two infrared sensors and PID control to follow a black track.

The project involved custom chassis design, circuit fabrication, motor control, sensor integration, PID tuning, and testing across straight-line, curved-path, and boundary-detection conditions.

[View Case Study →](https://anshiamy.github.io/projects/line-following-robot.html)

---

### VEX Toy-Sorting Claw Machine

An autonomous VEX robot that sorts objects into selected category areas using touch input, colour detection, gyro-assisted turning, distance sensing, and a motorized claw-and-lift mechanism.

The project focuses on mechanical design, modular C++ control logic, sensor calibration, autonomous navigation, and team-based system integration.

[View Case Study →](https://anshiamy.github.io/projects/claw-sorting-machine.html)

---

### The Shifting Grove

A Python and Pygame fantasy maze game featuring procedural maze generation, BFS fox pathfinding, three integrated mini-games, reward-based progression, local multiplayer, and custom user-interface transitions.

The project demonstrates object-oriented programming, modular architecture, state management, game AI, collision detection, procedural generation, and debugging across a larger software codebase.

[View Case Study →](https://anshiamy.github.io/projects/the-shifting-grove.html)

[View Source Repository →](https://github.com/AnshiaMY/The-Shifting-Grove)

---

### Embedded Car Dashboard

A Raspberry Pi Pico-based dashboard prototype with Python-controlled LEDs, button inputs, a potentiometer-controlled OLED fuel display, and a temperature-sensitive warning indicator.

The project combines PCB fabrication, soldering, GPIO and analog input mapping, embedded programming, wiring, continuity testing, and hardware debugging.

[View Case Study →](https://anshiamy.github.io/projects/car-dashboard.html)

---

### Copper Trace PCB Build

A focused hardware mini-build documenting the fabrication of two copper-board PCBs.

The process included trace transfer, etching, drilling, soldering, continuity checks, and powered LED circuit testing.

[View Mini Build →](https://anshiamy.github.io/projects/pcb-fabrication-mini-build.html)

## Running the Website Locally

The portfolio is a static website and does not require a build process or package installation.

Clone the repository:

```bash
git clone https://github.com/AnshiaMY/AnshiaMY.github.io.git
```

Open the repository folder and launch `index.html` in a browser.

For the best local-development experience, the site can also be opened with a local server such as the VS Code Live Server extension.
